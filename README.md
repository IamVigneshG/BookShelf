# BookShelf

Nice app to learn android basics

Topics Covered:
	1. Recycler View,
	2. Room Db,
	3. Handling Adapters,
	4. Navigation Graph,
	5. Data Model Class,
	6. State Flow,
	7. MVVM architecture,
